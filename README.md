# YouTube Metrics CLI

CLI-приложение для анализа CSV-файлов с метриками YouTube-видео и формирования отчётов.

## Описание

Приложение читает один или несколько CSV-файлов с данными о видео и формирует отчёт по заданным условиям.

Реализован отчёт **clickbait** — он находит видео с:

* высокой кликабельностью (CTR > 15)
* низким удержанием (retention_rate < 40)

Результат выводится в консоль в виде таблицы.

---

## Формат входных данных

CSV-файл должен содержать следующие колонки:

```
title,ctr,retention_rate,views,likes,avg_watch_time
```

Пример:

```
Я бросил IT и стал фермером,18.2,35,45200,1240,4.2
Как я спал по 4 часа и ничего не понял,22.5,28,128700,3150,3.1
```

---

## Установка

```bash
pip install -r requirements.txt
```

Минимальные зависимости:

* tabulate
* pytest (для тестов)

---

## Запуск

```bash
python -m app.main --files stats1.csv --report clickbait
```

### Параметры

* `--files` — список CSV-файлов
* `--report` — название отчёта (например, `clickbait`)

## Архитектура

Проект разделён на несколько слоёв:

* `reader.py` — чтение CSV-файлов
* `models.py` — модель данных
* `reports/` — логика отчётов
* `formatter.py` — вывод таблицы
* `main.py` — CLI-интерфейс

### Расширяемость

Для добавления нового отчёта:

1. Создать класс в `reports/`, унаследованный от `BaseReport`
2. Реализовать метод `generate`
3. Зарегистрировать отчёт в `registry.py`
